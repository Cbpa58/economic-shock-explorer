# Economic Shock Explorer

An interactive web application that explores how major economic shocks
(e.g. COVID-19, interest rate hikes) impact key macroeconomic indicators.

## Features
- Time-series analysis of macroeconomic data
- Interactive charts
- Clear economic interpretation

## Tech Stack
- Python (FastAPI, pandas)
- React
- Public macroeconomic datasets
